# Assets Folder

## large-image.jpg

This folder should contain a `large-image.jpg` file for testing purposes.

### To add the image:

1. **Option 1: Use a placeholder service**
   - Download a 1200x800 image from https://placehold.co/1200x800.jpg
   - Save it as `large-image.jpg` in this folder

2. **Option 2: Use any large image**
   - Place any JPG image (at least 1200px wide) here
   - Rename it to `large-image.jpg`

3. **Option 3: Generate via command line (macOS)**
   ```bash
   curl -o public/assets/large-image.jpg https://placehold.co/1200x800.jpg
   ```

The image is used to test:
- Largest Contentful Paint (LCP)
- Page load performance
- Browser rendering metrics
